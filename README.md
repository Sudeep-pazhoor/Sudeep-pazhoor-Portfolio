# Modern Portfolio Website - Sudeep PS

Personal portfolio website built with React. It showcases my profile, experience details, and contact information. Includes a simple contact form for reaching out if required.

## Design Features


## Tech Stack

- **React 19** - Latest React features
- **Vite** - Lightning-fast build tool
- **GSAP** - Professional-grade animations
- **React Icons** - Beautiful icon library
- **CSS3** - Modern styling with custom properties

## Installation

```bash
# Clone the repository
git clone https://github.com/Sudeep-Pazhoor/Portfolio.git

# Navigate to project directory
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Sections

1. **Hero** - Bold introduction with gradient text and CTA buttons
2. **About** - Professional summary and personal information
3. **Skills** - Animated marquee showcasing technical skills
4. **Projects** - Featured work with hover effects and links
5. **Experience** - Professional timeline with current role highlighted
6. **Education** - Academic background with glassmorphic cards
7. **Certificates** - Professional certifications and achievements
8. **Contact** - Multiple ways to connect with social links
